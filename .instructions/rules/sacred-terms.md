# Sacred Terminology — Canonical List

## Purpose
Single source of truth for terms that must NEVER be translated, even in Spanish documents.
All other governance files reference this file instead of maintaining inline lists.

## Core Concepts
- Problem Statement
- Context Engineering
- Context Debt
- Context Drift
- Build-First Bias
- Speed Theater
- Discovery
- BMAD
- Exit Criteria
- Gate Review
- Three-Move Pattern

## Artifacts
- Context Document
- Signal Log
- Context Update Record
- Iteration Brief
- Solution Brief
- Story Files
- Decision Log
- Build Validation Report
- KPI & OKR Register

## Roles
- Context Engineer
- Product Lead
- Tech Lead

## Anti-patterns (proper names)
- Over-Context
- Silent Agent
- Frozen Context

## Casing Rules
- Always Title Case as shown above
- Never italicize, never quote — they are naturalized terminology
- In Spanish text, Spanish articles agree with grammatical gender:
  el Problem Statement, la Solution Brief, los Story Files, los Skills

## Violation
- **Severity**: Alta
- **Action**: Auto-correct immediately (casing, translation attempts, quoting)
- **Log**: Record correction in changelog entry
